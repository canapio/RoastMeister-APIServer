# RoastMeister-Server
This application is server application for Roast Meister(iOS) based on Node.js


## Prerequisites
* [Node.js](https://nodejs.org)
* [MongoDB](https://www.mongodb.org/)

## Getting Started
```bash
# Start the mongo server in a separate terminal
$ mongod

# Run server.js
$ node server.js
```
